開源IDE-NetBeans 6.1

5/4/08

NetBeans IDE 6.1
http://www.netbeans.org/



支持的編程開發技術不少：
Ajax、C/C++、Databases、Debugger、Desktop、Editor、GUI Builder、Java EE、Java ME、Java SE、JavaScript、Mobile、PHP、Profiler、Refactor REST、Rich Client Platform、Ruby 、SOA、SOAP、UML、Web、WSDL、XML…

NetBeans IDE 是一個為軟件開發者提供的自由、開源的集成開發環境。您可以從中獲得您所需要的所有工具，用 Java、C/C++ 甚至是 Ruby 來創建專業的桌面應用程序、企業應用程序、web 和移動應用程序。此 IDE 可以在多種平台上運行，包括 Windows、Linux、Mac OS X 以及 Solaris；它易於安裝且非常方便使用。（NetBeans.org中文)

5.5號還是NetBeans Day呢！

這個假期過得不算太壞，但是感覺過得太快！幸好是雨天，適合當”宅男”。

我不喜歡寫Program，但是愛好看一些與程序設計的書；還記得10年之前用Basic或Foxpro在Win系統下玩玩而已；近期要讀的東西不少：

《程序員（07年精華本）》沒有看完的部分終于在假期第一天被讀完，頁面上做的筆記到處都是的，其實很多知識只是初步了解一下。閱讀此品牌雜志的目的，也在于關注新技術和網絡新應用，不至于自己還是井底一只蛙。

《Mashup Web2.0技術——基于Amazone.com》這是5.1前的一周買的，但還沒有開始讀。”混搭MashUp”的網絡應用實在是太豐富了，它究竟涉及哪些技術？如何把各種API更好地利用起來組成個性化網絡生活？我只是暢想把它讀完之后會有點收獲。

《C語言開發入門與編程實踐》，飛思科技的教材很厚啊，因為它的開發環境是OpenSource的Dev-C++打造（是BloodShed.net發布的一款C/C++的IDE），所以與一般的講Turbo C++、Visual C++的教材相比有獨到之處。

《丑陋的中國人》，因為柏楊(原名：郭衣洞）的逝世，讓我一定要買本他的書來珍藏，何況本人很佩服那些善于剖析人性的有識之士。再來回味他說的一句著名的話：我們的丑陋，來自我們不知道自己丑陋！

4月 29 2008

國旗-你能為他們降下來嗎

4/29/08

有人問：”家樂福，你還搜得到嗎？”當然能啊！

雖然百度不能找到網頁，雖然“中國人沒有理由使用百度百科”，但是它的貼吧還是存在的。我用Google.com還找到了曾經出現過的“受賄門”。我不想再提”紅心”事件了，就此打住吧！

我也算是眾多網民之一了，其實網民不是什麼，只是有些簡單的想法能上網就行，能夠傳遞自己公正的話語和傳播公開的信息就當然是最期望的，不想往自己的臉上加貼標簽讓自己成為精英或者是”意見領袖”，那些也毫無意義！

昨天、今天還有幾件事情受到全社會的關注，那些對表象憤怒的人面對國內受難的逝者又會以怎樣的行動來表示呢？在看到：因無可控制的病毒奪取幼小的生命、無法預知車禍中的遇難者、被販賣至東莞的黑工的時候，我們對一些擔當的管理者、責任人為什麼不憤怒？！反對西方媒體的不實報道，我們的人可以舉起旗幟以示愛國和尊嚴，但對因人為原因造成的重大安全事故中失去生命的國民，我們何不降下國旗在致哀的同時給予世人更大的覺醒？

最後，祝願活著的人多保重！為那些逝者致哀！
