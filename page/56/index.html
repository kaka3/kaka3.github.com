搶的是尊嚴

Wed, 3 Mar

這不是影評，大部分是摘錄了其中的人物對白。
看《錦衣衛》的時候，因為DVD碟片的聲道問題，所以影響了觀看的耐性。
錦衣衛里面的四個高手強人：朱雀，白虎，青龍，玄武。
“完成任務是錦衣衛唯一的尊嚴！”

與脫脫的拼斗，大漠判官的飛天旋轉雙刀的確耍的不錯。
吳尊由強盜變成了盜亦有道的大漠判官。

無露不成戲，在《錦衣衛》里趙薇，甄子丹，徐子珊都也是身段不錯啊！
皇帝的荒淫奢侈，脫脫的背紋，青龍與喬花澡池隔墻對話。

青龍：在我完成任務之前，誰擋我，無論是朋友還是敵人，他將會先倒下！

青龍與脫脫同歸于盡，是悲劇了；江湖有江湖的規矩；
正義鏢局頭頭的女兒喬花最后完成了交回傳國玉璽的任務；
喬花結尾說：有希望是幸福的！

親王爺（脫脫的義父）的一句話：我最恨的是算不清的帳。

白虎/朱雀與玄武之間在牢室的決斗很精彩。
場面火爆的地方還有：
森林中的廝殺，山谷中的石陣，古堡上空的轟炸，火槍隊的滅亡，
殘廟中的決斗，大漠判官與脫脫之間的比試……
吳尊手里拿的是火箭彈還是箭筒？？明朝時候的武器也太先進了啊！

賈精忠對玄武的一句話：當人們看到你成功的光芒后，就會忘記你手段的黑暗。
自然也是讓人笑翻的一句話。

喬花：這個世界真的變了嗎？我們老百姓真要活得這么沒尊嚴嗎？
青龍：尊嚴不是別人給的，是自己爭取的。

青龍：我從來不問因由，只聞時間地點何人。
喬花：你想退出嗎？
青龍：錦衣衛本來就是一條有進無退的路，這條路很短暫，我很快會走完。

喬花：路只要一直走下去，一定會海闊天空的。

港片中常見硬幣，這次《錦衣衛》也見硬幣，是大漠判官誓言要在硬幣轉停之前將青龍打倒，此為“英雄真是不打不相識啊”！

判官對青龍說：天鷹幫家訓，有天我能找的值得用自己性命去搶的東西時，
我們就由強盜變成盜亦有道了。
判官問青龍：你要搶的是什么？
青龍思慮良久：尊嚴！！

3月 3 2010

新本本的折騰

Wed, 3 Mar

決定很久的目標，終于下手了，歡迎新的本本成為我的又一件裝備吧！

2月28元宵節，逛了世博電腦城，電腦的品牌或價格都不算符合我；也去國美/蘇寧等看了，百佳旁邊的那家國美店不知道為什么那天正在拆柜臺；最后還是去時尚選了新本本：HP-CQ511 VX579PA#AB（產品號碼），送了電腦包，鼠標，還有一張床上用的電腦桌。付錢的時候小MM用點鈔器差點多要我100元，最后還是我手點的正確，唉，機器有時候還是不能代替人工啊！但看看平時自己買的手機和相機都是在這里買的，這也是一種Feel吧！

預裝的是Rad Flag Linux Desktop 6 (sp1)的操作系統；因為用了好幾年的win系統，所以想換一下口味，說真的KDE比WIN的GUI設計真的很亮麗。于是這2天就折騰起了紅旗桌面版6.0，在“紅旗人”社區逛了許久，也下載了PDF用戶手冊，還是感覺前面一些茫然。

在DESKTOP 6中，Firefox還是2.0版本的，想讓它升級到3.6，但是試了幾次只能從root下找到js（shell）文件后才能使用；不知道我下載的文件解壓縮是不是正確的。

輸入法的問題，自帶的SCIM，用智能拼音/全拼雖然很不錯，但還是與谷歌/搜狗在windows下的拼音輸入法有差距，限制了我的打字速度啊。

還有就是瀏覽器VPN代理的問題，免費VPN中我比較常用PacketiX和Puff，雖然前者有Linux版本，但是需要自己編譯而很難配置成功；后者在win系統中運行很簡單的，但是要安裝在Red Flag Linux下，找了網上的資料說下載wine后可以進行puff的安裝，也許值得嘗試吧。

還有個問題就是，Red Flag Linux桌面版在開機和關機的時候，很需要時間，也就是速度都不很理想。

基于上面這個速度的原因，所以又拿出好久之前申請到的Ubuntu 6.06 LTS盤，安裝過程很快，大概10分鐘左右成功啟動了。雖然Gnome的GUI表現也很不錯，但是屏幕及字體太大，聲卡也不能用了，估計要安裝顯卡聲卡等驅動才能解決。另外，6.06中的Firefox還是1.5舊版本的；系統中沒有中文輸入法；然后，重新啟動后，時間總是要去調整一下，就這幾個讓人頭暈了。

所以試用了Ubuntu不久，想一想還是回到Red Flag Desktop 6，再安裝后才知道時間花的更多，估計讓我等了個把鐘左右，隨機自帶的DVD安裝盤包括了驅動程序，我想也許只有這個原因造成的。

于是，現在遇到一個系統迷們爭議的問題：裝Linux好還是Windows好呢？
