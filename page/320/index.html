6月 4 2007

巧遇《Ubuntu官方指南》

6/4/07

上次在自己的blog上转了一本书的内容，那本的名字是《Ubuntu从入门到精通》；以6.10版本(Edge Eft)系统为操作平台；

星期天照例顶着烈日去购书中心叹冷气；从98->XP->Vista的倒是琳琅满目,我看着Win系统的书却头晕；碰巧在操作系统一类的书架上看到有3本《Ubuntu官方指南》静静地放在那里！而这家很大的购书中心，摆放GNU/LINUX操作系统的书，有少得可怜的几本Debian,SuSe,Rad Hat(Fc5),另外就是鸟哥(Vbird)的Linux私房菜系列写的很不错！其他的几本Unix,Free BSD,Solaris都是放在很偏僻的角落！Unix类我是没有指望去用，但是Linux的还是可以试试的,现在就是期待”乌班图”Ubuntu!

看了一下这本Ubuntu的书，5月份出版的，觉得有些也不错，主要还是适合我申请的那套免费CD;这个6.06 LTS(Dapper Drake)版本可以支持到2009年，即使最近有了7.04版本(Feisty Fawn)；

5位著作者之一的Mako曾是Ubuntu团队的创始者和Canonical有限公司第一批员工中的一员;现任职于MIT媒体实验室,参与OLPC(100美元笔记本电脑)项目;

去年的8月11日是The Official Ubuntu Book英文出版物诞生的日子！

内容简介:

本书是Ubuntu第一个长期支持(Long-Term Support)版本的官方指南。

本书首先介绍了Ubuntu的历史、Ubuntu及其相关项目Kubuntu的安装、使用和定制，然后介绍了在实际使用Ubuntu的过程中具体问题的解决方案，同时还介绍了Ubuntu Server这样的高级主题。此外，本书还讲解了Ubuntu的社区和文化，使读者能够对Ubuntu有更加深入的了解。本书最后介绍了Ubuntu相关的项目和命令行的使用，具有很高的参考价值。

本书是官方认可的Ubuntu使用指南，适合广大Ubuntu用户阅读。初、中级用户通过阅读本书可以全面提高其使用技能，专家级用户可以立刻应用书中介绍的技巧。本书也可以作为培训机构的Ubuntu标准培训教材。

6月 2 2007

林肯公园Minutes To Midnight

6/2/07

看到上面这个标志相信不少朋友都会认出所指是谁，那就是Link Park(LP)令肯公园组合！

5.15日正式发行了他们重生后的最新专辑《Minutes To Midnight》！

但是这张专辑出版之前都有好多地方在介绍和评论，在豆瓣里见到有8人给的是差评；而另外一个音乐社区Last.Fm里面试听的人数目前仍然排在首位；

其实在发行之前就有主打歌”What I’ve Done”还在电台HitFm887里热播；我在听到这首作品后，不久便在市场上看到了这张专辑的CD；可是听到的”What I’ve Done”跟电台播的旋律大不一样，CD里面的听起来的乐感太缺少强劲有力的嘶吼声；幸好有Shadow Of The Day来代替我的埋怨！

在逛Youtube的时候，突然遇见有What I’ve Done的视频资料，大概有几种版本的；一一试听完后，才觉得下面这段是比较符合我第一次在听电台时候找到的震撼感觉!

Link Park在这张专辑中更加突出了反映了政治.反战.环保.灾难等世界问题,可见一支乐队的改变也开始从简单的音乐层面向社会生活的深层次去发掘灵感；尽管会招致有些人的反对，但作为LP的Fans们需求的是6位狂放音乐人的灵魂和至上的旋律；无论是重金属或Punk，还是Hi-pop或Rap,只要动听入耳，也许就足矣！！

酷热的六月，下班之后，放大音响，让What I’ve Done刺激我的耳膜，让流淌的汗水带走心中的郁闷！

关键词：收购和Gears.Surface等

5/31/07

要到6月了，意味著07年快过一半啦！Google和微软两大巨头也忙着推出他们的新服务或产品，或者忙于收购其他国内公司来扩充自己的业务！

这样的事情好是好，但也难为了那些追踪新鲜爆料的Geek或blogger们啊！我虽称不上甚么Geek之类，但自己订阅的rss也不少，看了很多新科技新信息就只有劳驾收藏夹来帮忙哦！

Google和微软在这几天发布的产品和网络新闻真不少！

先说Google的吧，也许有的是我不太感兴趣而引致些许遗漏的，但不要紧，毕竟这里记载的是我自己的所见所闻而已！

1)Google收购西班牙照片共享网站Panoramio

这项服务在使用Google地球、地图产品的时候就可以发现的；如果不熟悉的话，还可以看看之前Willamlong所发的”照片共享服务panoramio和flickr的对比评测“一文!

2)Webleon称Google Gears为”互联网的离线发动机“,首先推出的应用就是离线的Google Reader;可以想见今后说不定还有离线Gmail,如同微软的email桌面工具Windows Live Mail Beta;

详情见正式页面:Google Gears (BETA)

(从全搜索的关系数型据库中实现本地存储数据)

Run asynchronous Javascript to improve application responsiveness
(运行异步脚本来改进应用程序的互动)
