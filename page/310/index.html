使Firefox速度提高4倍的秘笈

7/28/07

Firefox比之IE的優勢在于其定制靈活，有豐富的皮膚和插件，從最初的1.5到現在2.0版本，都覺得不錯，盡管以前還嘗試過Flock.Opera等非IE內核瀏覽器，但在我的電腦上常駐常用的還是Firefox—火狐。

關于提升它的速度問題，以前也有過這方面的很多巧門；今天又看到其他人試用過的另外一種方法，據說是可以將速度挺高4倍；反正有人用，那就咱也來試一試吧！

首先，在URL栏输入“about:config”， 将打开一个参数的命令行清单，这就是我们以下进行编辑的第一步。

第二步，在下面的清单中找到“network.http.pipelining” 然后双击“network.http.pipelining”将其值改成“true” 下面我们要做的就是找到“network.http.pipelining.maxrequests”并把它的值改的高一些，如果你改成40，意思就是要求它每次发送40个请求。

最后一步就是在Firefox任何一个地方点右键，选择“新建”—“整数”在弹出的对话框中输入 “nglayout.initialpaint.delay”并将其值改为“0”，表示当浏览器收到网站回复的信息后能够快速的反应。完全关闭 FireFox后退出，再重新打开FireFox，你将会有不一样的速度体验，会有3-4倍的速度提升，有兴趣的朋友可以实验一下。

好的电脑可以更快 效果更好。

兔斯基

7/28/07

“兔斯基(TUSKI)”,我居然用谷歌拼音能夠把這個單詞打出來，它還是一個流傳網絡間的詞匯呢，其風頭甚至賽過昔日的“洋蔥頭”；偶然從改版后的新浪論壇瀏覽到“銳話題”，不成想卻發現了一份有趣的試卷—-2007“兔商”等級考試。除了“智商”.”情商“.”健商”等之外，也許你和我一樣頭一次聽說還有一個“兔商”的詞吧？

下面是試題及答案，看完你就明白屬于“兔斯基”的一些有趣故事了！

一、选择题（共5题，每题4分。）

1 兔斯基为什么叫这个名字？(答案C)
A 兔斯基是俄国籍兔子。
B 兔斯基的作者受大片中“崔永元夫斯基”的影响。
C 作者受短篇小说《请用“S”拼我的名字》的影响，迷恋“斯基”。
D 因为作者的名字叫兔斯基。

2 兔斯基究竟是什么？(答案A)
A 喜欢胡思乱想，满腹牢骚的兔子。
B 流氓兔
C 自杀兔
D 团圆兔

3 图中的这个表情是什么意思？(答案D)

A 抽筋
B 悲痛难当
C 抒情
D 得意忘形

4 选项中哪个不符合兔斯基性格？(答案C)
A 颓废
B 闷骚
C 笨蛋
D 放纵

5 现在兔斯基系列有多少个表情？(答案B)
A 20多
B 40多
C 50多
D 100多

二、填空（共5空，每空5分。）

1 作者王卯卯的“卯”字拼写成＿Mao＿，王卯卯的性别是＿女＿。
2 兔斯基的性别是＿男＿，婚姻情况为＿未婚＿，身高约＿1.2M＿。

三、判断题（共2题，每题10分。）
1 兔斯基边开口说话，边用柔软若面条的手臂和身体的扭动表露自己的心情。
(错,兔斯基从不开口说话。)

2 兔斯基喜欢喝牛奶，吃奶酪，不喜欢吃胡萝卜。
(正确。)

四、名词解释 （共3题，每题5分。）

1、兔斯基主义：
像兔斯基这种思维方式，这种生活态度；

2、兔斯基：
一个小兔子用柔软若面条的手臂和身体的扭动表露自己的心情。有点懒散，有点颓废，时常茫然，容易失落，喜欢胡思乱想，经常满腹牢骚。

五、问答题（共2题，每题10分。）

1、为什么兔子平时没有嘴巴呢？
张不开嘴巴，眯着眼睛，兔斯基的特征，源于为他塑造的性格，以这种有点懒散、有点颓废的性格基调下，他的言行举止似乎理所应当就是这个样子了。 受到强烈刺激、震惊的时候，兔斯基偶尔会把眼睛睁开，嘴巴张开，作嗔目结舌状。

2、你愿意用哪些词来评价这只兔子呢？
仁者见仁智者见智，此题无标准答案……

方便面涨价

7/25/07

人的一生，就是吃喝拉撒.衣食住行的一生吧！房子.汽车等等都不用说了，近看我们百姓的日常吃喝都有些快支撑不住了。说是大家的收入比以前提高了不少，可是这物价每年上涨的步伐可是一点没落下。

昨天一起床就听到电台的消息说方便面也涨了！本来我也拒绝这种无多大营养的食品，但是一听”涨”字就会与高价猪肉联系起来！小时候穷的很，谁家能够吃上一顿猪肉包的饺子都算是富裕的日子，而现在我看，如果今后我能有一包方便面吃也是幸福的新生活吧！

方便面也成为热门话题，可以从百度搜索排行榜中反映出来；看看上升最快top50里面，打头阵的搜索关键词居然是“方便面涨价”！谁能猜猜哪些人最关注”方便面“的价格问题呢？

在这个榜单上还看到《十面埋妇》也被收录啦！这可是近期在博客中传得很红的一部DV视频，亦是博客名人三表叔的作品。有点纳闷的是为什么要与腾讯QQ联手播放呢？我这儿是上不了QQ Live的啊！他那么讨厌新浪，可人家还有网页来夸赞《十面埋妇》在版权上的积极态度。
