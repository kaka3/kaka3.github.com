ms will end some winos support

Thu, 25 Feb

看來現在配置新電腦且想安裝WIN系統的話，還是直接安裝WIN7啦!

2010年4月13日：終結對Windows Vista RTM的支持；
2010年7月13日：終結對Windows 2000所有版本和Windows XP SP2的支持。

via：windows team blog

引用部分原文：

If you or someone you know is running Windows Vista with no services packs installed (also known as Windows Vista RTM or SP0), please use Windows Update to install the latest service pack available – Windows Vista Service Pack 2 (note you’ll need to install Service Pack 1 first). We also recommend checking out our latest Windows release – Windows 7. On April 13th, 2010, Windows Vista RTM will reach the end of support.

And on July 13th, 2010, all versions of Windows 2000 and Windows XP SP2 will also reach the end of support. We recommend those of you still on Windows XP SP2 to install the latest service pack available via the Microsoft Update website – Windows XP Service Pack 3 – or look at moving to Windows 7. Folks still on Windows 2000 will need to look at moving to a more current version of Windows, such as Windows 7.

折騰的春晚

Wed, 24 Feb

唉，還是來說說看電視的一些事兒吧！
春節不是我折騰電視機，就是我被電視節目折騰了。

這次回家過年看電視，發現較大的變化就是多了鳳凰衛視臺，還有一個CSPN體育臺，如果不是它們陪我打發無聊的休假日子，我不知道這個春節會過得怎樣。正當CCTV5直播冬奧會之際，有幸從CSPN欣賞了NBA全明星的賽事，看到納什捧到了3分王的MVP獎杯，還有東西部之間的對抗賽；而且還在深夜觀看了一場英足總杯第五輪中的曼城VS斯托克城的賽事，實在是要多謝CSPN體育聯播這個平臺。

央視春晚在大年30還是要瞄一眼的。現在腦海里只有“老虎隊”們的影子，《愛》，《青蘋果樂園》，《蝴蝶飛呀》等3首經典依然可以回響在耳邊；是他們喚起了我從初中3年級起瘋迷他們的舞姿和歌聲的青春片斷回憶。

回到老家的那天剛好是臘月24日，到了晚上的時間按照慣例應該是本地電視臺推出迎春晚會的時刻，自然今年也不例外；我想從頭到尾靜靜看下去，但手指還是耐不住要轉臺，我不相信全縣的百姓們對這場晚會的品味會如此之低。聽說某姓戴的女歌星來本縣表演了；而且那晚看了一個個人小品《時髦老太》，真是差點讓人笑掉大牙；主持人還振振有詞稱其為老師；在我看來也許是小縣城吧，拿得出來的節目也就只能這般水平了，不過老鄉們倒是對“嘉年華”酒店的名字可能還能念念不忘的啦，看來小地方官方機構的宣傳效應不可低估。

也許對老爸來講，他沒有時間和心情去欣賞這2場晚會，算是一種小小的遺憾。但在聽到我說不好看的評價后，他也只能在來家拜年的客人面前同意了我的觀點，春晚越辦越不好看了。

可為什么還每年要花巨大的人力物力財力去操辦呢？百姓的精神文明建設在每屆春晚之后也不見得有所質量上的提升啊，相反，近些年來鄉村里到處是麻將聲聲入耳，而且農村里的環境污染也是愈來愈嚴重。為何不把辦晚會的精力和金錢投入到改善農村的生活面貌和建設一些便利的民生工程呢？

央視春晚是大舞臺，大制作，主旋律的，和諧的，主持人是盛裝出鏡的；
縣城晚會是小舞臺，小制作，半娛樂的，外來的，主持人的面目已淡忘；

兩臺晚會，就這樣折騰了數年，也折騰了無數人。

留給有意識的人們去思考：為什么春晚統治我們那么深？

buzz in gmail

郵箱中的微博:Buzz

Buzz的主頁: http://www.google.com/buzz ，這也是一項社會化網絡（SNS）的應用，只不過是跟郵箱關聯起來的。

如果你使用QQ Mail或Foxmail的話，就可以注意到在“閱讀空間”里有“廣播”這項功能；那么，Buzz的部分功能也跟這個“廣播”有些類似，在郵箱中與好友進行信息分享和互動的活動。

大約在2月16日，我登陸Gmail查看郵件的時候發現里面多了一個Buzz的項目，相比這個日期其實2月10號之后已經有很多G粉已經迷上了這個輕量級的應用，紛紛發出了自己的試用體驗以及一些高級技巧，還有一些小貼士也不可錯過。

然而，又有信息說，Buzz這種發布方式涉嫌泄露個人隱私，所以，有人愿意用，也有人選擇關閉了這個微博功能。自然地，對于大陸網民之一的我也擔心，開通了類似Twitter微博的功能后，Buzz會不會像Youtube/Picasa/blogspot等這些服務遭遇GFW的“照顧”呢？！

在我簡單試用一下后，感覺TA就是個FriendFeed和Twitter的結合體，前者可以集成個人的lifestream，而后者是即時發布自己的動態信息，Buzz的功能則是兼而有之，并且TA對字數沒有限制。目前，在Google Profile中也可以看到自己的Buzz更新信息。如果要關閉TA，可以在Gmail的底端選擇”turn off buzz”，或者到“Settings”選項中進行自己的設置。

Gmail從04年愚人節開始創造了互聯網業界的一個神奇，而Buzz會不會在2010年給Gmail帶來另一個神話還有待觀望。

在信息過載的網絡時代，用還是不用Buzz的確是個問題。目前我的做法是：從Gtalk發布信息到FriendFeed，然后讓FF的信息更新到Twitter，再接著就是讓Buzz納入Twitter上的內容，這樣就是Gtalk->Buzz之間在單線對話了。沒有辦法啊，誰叫我只用一個IM客戶端呢？！
