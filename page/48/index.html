升級一下“酷龍”

很久沒有上et8.net了，
也就是說我有很久沒有把K-Meleon CCF升級一下。
先前使用的版本號是：0.09 Btea 4 v1，
今次是下載了0.096.5 Pre版本；

嗯，每次看TA的版本號簡直是要進行一次“頭腦風暴”，升級之前不免要對比之前的版本，實在是有些難以忍受；但，我追求的是TA的打開頁面速度，畢竟比Firefox還要略勝一籌，至少在這臺舊機器身上發揮得還是不錯。

使用“酷龍”有好久了，有些舍不得放棄；在win系統下，要找到一款稱心的軟件，有時候不是靠廠商的推廣，而是憑自己的feel。

Public測試版下載地址，請點擊此處查看。

2月 26 2010

500 server error on friendfeed

FriendFeed杯具了

500 Internal Server Error

nginx/0.6.34

這是剛才開啟Puff后，再次訪問FrienFeed后反饋的結果，杯具發生了，唉！

意外時時有，但這次，我寧愿沒有經歷過。

下午15:30前后的樣子，想用Gtalk更新FriendFeed和Twitter都沒有成功，起初只是看到機器人ff@chat.friendfeed.com處于下線狀態，當然不能接受我發出去的信息了；接著，通過YR讀到了TechCrunch上的一篇消息，反映了在Twitter上僅僅有50條Tweets在它down機的30分鐘內出現。這不得不令人疑慮，是不是Facebook 收購（2009年8月10日）FF后看到沒有多少用戶使用了，于是就想關閉服務呢？

我一直沒有放棄FriendFeed，她是個挺不錯的社會化網絡服務，提供了豐富的內容聚合，而且是前Google工程師們開發出來的，UI界面很清爽，blogger們對她的“高明之處”亦有不錯的贊譽之言。

用Google.com搜索關鍵詞FriendFeed，可以得到34,200,000條相關條目。

cryptome is back after closed claim with microsoft

蚍蜉撼大樹

一場不大不小的風波啊！呵呵！

在國外發生的事情如果在國內發生會有什么結果？？

http://cryptome.org/

很NB的hacker網站啊！從CN的網頁上看，居然有內地和臺灣的機場圖片，太原和西昌衛星發射基地的圖片（當然它們是Google鳥瞰圖的截面），還能瀏覽到北京奧運會火炬保護活動的照片，也有一些512地震的圖片在上面找得到。看來，這里早已經成了一篇秘密之地。

“小小的“Cryptome.org如果真的跟軟件巨頭微軟叫上勁的話肯定會吃不了兜著走的，想一想曾經倡導“Freedom“精神的“海盜灣”的處境吧，蚍蜉撼大樹真的很難。但是，這次微軟的態度簡直來了個180度的大轉變，已經不再對其在網站上泄漏微軟“全球犯罪調查手冊”（GCCH）一事進行追究，而且還要求“恢復網站，并且不再要求刪除該文件”。

之前，微軟以DMCA（數字千年版權法案）投訴其ISP（Network Solutions），認為Cryptome在網站上提供GCCH下載地址的行為未經微軟授權，違反了DMCA的保護原則，并要求刪除鏈接地址，隨后網站面臨被封停的處境。

下面的郵件證實ISP已經在收到微軟撤訴的郵件后恢復了Cryptome的訪問。
