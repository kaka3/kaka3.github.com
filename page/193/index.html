我的QQ年齡7.1年

11/12/08

11月11日，很奇妙的日子，連11位明星都摻和到“聖光棍騎士團”了，一幹人等不亦樂乎啊！

當然了！我們還應該記得：11月11日也是一戰停戰90周年的日子。

無獨有偶，在這般寓意多多的日子裏還迎來了騰訊10周年之慶。通過官網頁面可以瀏覽諸多Q友的10年故事，並且發現了自己在QQ上的注冊時間（2001年09月29日），QQ年齡已經7.1年啦！記得IM工具最先使用的是ICQ呢，後來才跟風注冊了個QQ號，於今我只專注於Gtalk的使用，時間真是過得太快了！而技術上是“一浪更比一浪高！”

1999年2月，騰訊QQ誕生。2001年2月10日，騰訊QQ在線用戶成功突破100萬大關。QQ最新更新是：QQ2009 Preview3 Build450最新版公布，優化了查看個人資料界面。

最近，騰訊與競爭對手（是51.com，還是51job.com？）之間的新聞也有2條抓住了大家的眼球，源於離職員工跳槽被騰訊起訴，其實在國外也有同類事件，如：前IBM高管跳槽蘋果被判違反競業禁止協議。

Gtalk與FriendFeed的互動

11/11/08

FriendFeed是個人信息聚合平臺，算是SNS范疇的應用吧。每次查看FriendFeed頁面信息的更新，總是會奔首頁而去，覺得等待加載頁面的時間有些麻煩。不過現在可方便了，可以通過Gtalk客戶端來實現與FriendFeed的互動，比如：發送個人信息，接受和評論等。我們所做的就是出除了熟悉幾個操作命令外，還要在個人設置頁面激活相關選項。

官方消息：Instant FriendFeed
（Notifications and Posting over IM）

在“IM Settings”選項中激活Gmail帳戶：

在“FriendFeed Over IM”頁面找到命令句法：

在Gtalk中顯示FriendFeed的聊天機器人：ff@chat.friendfeed.com，其他命令：

1）@me text - post to your feed
2）@me - list entries in your feed
3）@room text - post to room
4）@room - list entries in room
5）@N text - comment on entry N
6）@N - show entry N
7）@user - list entries in user's feed
8）like N - like entry N
9）pause - Pause notifications
10）resume - Resume notifications
11）list - List recent entries
12）help - this command

其實不記得那麼多語法也沒關系啊，只需要記得輸入第12）個“help”就可以獲取即時幫助信息。

2）娛樂轉貼：【新史記之 趣論溫州追韓信】

溫州楊姓書記旅法蘭西而不歸，坊間哄傳，江湖嘩笑，而鹿城大窘焉。窘迫情急，忽出奇計曰“越洋追韓信”也，派使者三，懷錦囊而西，越大洋而追書記焉。抵巴黎，出錦囊一，欲與書記晤談，書記隱於市，晤面未遂；出錦囊二，遊說僑領誘其出，僑領等囁囁回避，又未遂；錦囊三，到此一遊而返也。

噫嘻！溫州之追韓信，其錦囊之計令人噴飯也乎哉！昔者，蕭何月下追韓信，乃韓信大才也，不追則為項王用，用則難克也；今之楊書記，大才乎？法蘭西必用其才乎？皆非也。如是則溫州之必追，恐另有難言之隱也：上必責以懈怠，致國威有損，此其一也；大員動輒去國，後來接踵仿效，此其二也；該員或有貪瀆，遠走外邦避禍，此其三也。有此三端，鹿城惶急，是以巴黎追韓信焉。對上則曰我已追之，對下則曰我已仁盡，自命乎上下周全也。

愚哉！鹿城有司！該楊謀定而動，策劃於內外，慎密乎始終，前有逃匿之官悠悠隱寓，後有絡繹之賊蠢蠢欲來，歸也乎？不歸也！是故溫州之勸返，明知無功而必追，矯情之蠢一也；所謂使者勸返，無非黨國大計，鄉梓殷殷之類，該楊貴為地方大吏，豈不知外崇內凶之實相？該楊乃洞悉實相而遠走，豈能再入爾等彀中？此又溫州勸返假情之蠢二也；該楊既走，追之何益？察其貪瀆，開其職缺，通報國際，一體擒拿可也，不思以法治其罪，依然弄權飾其非，此則溫州勸返可惡之蠢三也。

有此三端，則楊氏之後，再發滬上吏員之逃，接踵而奔避之賊官絡繹不絕者必也，有何怪哉？

（by：饕餮 凱迪社區|貓眼看人|知名網友）
