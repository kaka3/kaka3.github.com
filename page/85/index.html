Tue, 29 Dec

GSM加密算法被公布

在Mashable.com上看到一則消息:Encryption Code for 80% of the World’s Mobile Calls Cracked，大意是移動手機的加密算法被破解，將影響80%的手機用戶的安全。
據“中國計算機安全”的報道：

本周一，一位德国电脑工程师Karsten Nohl宣称他已经破解了现有手机GSM无线网络的解密算法。据纽约时报报道，Karsten Nohl此举的目的是提醒人们注意当前GSM网络的安全漏洞，GSM解密算法目前正被全球80%以上的手机所使用，它已经被手机网络使用了21年。

一年一度的黑客大会 Chaos Communication Congress近日在柏林举行，而Nohl则在这次会议上做了如上的宣布，他表示共有24位破解者参与了这次破解活动，破解出来的密码本数据量达到了2TB。

在今年8月份举办的一次会议上，Karsten Nohl曾宣布要对GSM加密算法进行破解。没想到这么快他便达到了目的。
