9/25/06

昨晚在明珠台Pearls里听到影片”Spider_Man”中的一句话:能力越大,责任越大!

9月 16 2006

Great mashups

9/16/06

地图(map)之软件或网络服务方面被我关注最多的当属Google Maps/Earth!
而把google maps与其他服务捣(Mash)在一起的应用算是很新奇的玩法,象flickr图片.Tuniu旅行.EEmap地标等等都为我们带来了面目一新的视觉享受;

“Mashup”已经流行起来,越来越多的服务都喜欢揉合在一块;而Google Maps API也是首发工具之一;关注Maps这方面的blog也是比较多的,名气显著的有:

  • Google Maps API Blog
  • Google Maps Mania (Mania跟fans的词意差不多吧)
  • Google Earth Blog
  • Ogle Earth
  • All Points Blog
  • ProgrammableWeb
  • The Map Room
  • Mapperz

    • 其中在Google Maps Mania这个站上可以看到很多mashups方面的应用,也介绍了不少的网站!
    涉及到很多类型的服务,有音乐方面的,寻人的,啤酒店,还有天气的.政治的,商业的,体育的,交通的,教育的,家居饮食,招聘求职,无线网络等科技方面的……总之,只有我们想不到的,没有做不到的;但发现国内这方面的创作项目很少很少!
