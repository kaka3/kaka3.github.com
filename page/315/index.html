6月 18 2007

父亲节

6/18/07

2007年的父亲节是6月17日，居然与端午节隔了一天时间；我好像记错了时间吧，上个星期码字的时候居然把第二周的星期日当成了父亲节，又闹笑话了；后来对照日历是自己错了,其实也不怪我吧，去超市的时候到处贴着广告，弄不准把日期颠倒的不只我一个人！

在父亲节和端午节之前的晚上终于给家中的老爸去了电话，发觉好久没有打电话问候老人家了，不免由衷生出一阵悔意；居然不知道几天前他受到意外的损伤，幸好立即去医院做了检查终无大碍；现在唯一希望的是他能够平安，快快恢复伤口；如果不是小妹在家里的话，估计他又会把这件可大可小的事情给”隐瞒“了；哎，做父母的时候他们的心真的很苦很苦！！

今天也算有点令人开心的消息，看到Google也识趣地做了个logo来庆祝父亲节这天的到来！

说起Google logo的首席设计师丹尼斯·黄,不少Google”粉丝”(“狗粉“？”谷粉“？）都耳熟能详，真没想到那些“涂鸦”之作都是在匆忙之中完成的，是不是太有才了?!

尽管早已功成名就，黄正穆依然冷静处事，与人为善。就是到现在，黄正穆在Google的办公室并不显眼。

一台拥有三个屏幕的大型电脑占据办公室的大部分空间，其他空间也就随便堆着放满东西的盒子。在电脑键盘旁，摆放着一个电子绘图板，这就是他用来创作的“杀手锏”。如今，黄正穆依旧担任Google的网管，管理一个大约30人的团队。他工作中的大约80%到90%的时间用于维护Google网页，每年仅设计大约50个Google徽标。令人吃惊的是，黄正穆毫不讳言，那些“涂鸦”之作通常都是匆忙之中完成的。不过，黄正穆的粉丝们认为，就算是那些设计堪称漫不经心，也足以令他们欢呼雀跃。

节选自Google图标作者：设计皆是匆忙完成一文

PS:用WLW(Windows Live Writer)发布日志的时候有些地方还没有完全适应，比如，我用Google输入法的时候，在中文/字母之间转换的时候老是反映慢慢地；还有开启这个软件的速度也不够快。甚至比GDS(Google Docs & Sheets）还慢？

山西窑奴.九江大桥.学生舞

6/16/07

仿佛越来越生活在压抑中，每天压力也不小；看到周遭的一切，很自然地变得让人不愿记忆正在发生的一切!而这不是自己愿意看到的状态！

1)就在”四川500余名初中生东莞实习每天被迫工作14小时“的事情曝光后，又一起童工案例继续了；由400位父亲泣血呼救而引发的”山西洪洞黑砖窑案“，让人看了很心痛也很愤怒；为什么在“和谐”的今天，还存在奴隶社会那样的剥削关系？受害者的权益根本都不存在了，还有那些黑心的骗子们，让他们早点遭到报应吧！

来自一位网友的的帖子让人看了十分难过：

农民工被欠薪，我们没有为他们呐喊，因为我们不是农民工；

一线教师被教师法抛弃，我们没有为他们呐喊，因为我们不是一线教师；
农民土地被强制征收，我们没有为他们呐喊，因为我们不需要种地；

沉默啊，沉默！
等到哪天不幸降临我们自己头上，谁来为我们呐喊？

2)北边的”山西黑砖窑虐待民工“的新闻仍处于焦点中，这下又看到报道说广东九江大桥被撞垮塌引发车坠人亡事故，真的很震惊!也许事故责任还在某些人的研究和查找中，但事故已经发生了，对于那些丧失亲人的家庭来说，他们的悲剧又该怎样收场呢？！

3)长沙教育局规定不跳集体舞者可能评不上三好生，看了这则新闻只能让人觉得制定规则的人纯属瞎整，又是要搞“大一统”和“一刀切”了；之前有说跳集体舞可防止学生早恋的，呵呵，很无聊的教育方式吧！我看当下的学生让其做个早操应付一下都是难哦！贫苦地区孩子们的教育情况他们有没有实地调查过？我们仍然处在应试教育的制度下，现在的孩子被这样那样的规定折腾着，还让他们怎么把精力投入到学习中啊?！

Microsoft Math 3.0

6/16/07

微软发布了新款计算软件:Math 3.0 (http://www.microsoft.com/math)
其实在windows系统中有一个附件的应用工具叫“计算器”，不过却是很少使用了；

而名叫Math的这款专业软件又有哪些亮点呢？

Microsoft Math 3.0主要特性：
1、图形计算器Graphing Calculator
2、向导式数学题解答Step-by-Step Equation Solver
3、方程和公式库Equation Library
4、三角计算器Triangle Solver
5、单位换算器Unit Conversion Tool
6、手写笔支持Ink Handwriting Support

可能对于学生和教师解答数学和理科题目带来了福音；之前，微软委托进行的一项独立调查显示，77％的老师和73％的家长都表示数学和科学是学生面临的最困难的学科，但只有36％的家长能够为孩子提供帮助。(信息引用)

当前售价19.95美元，可以下载30天的试用版。

PC机的最低运行配置和推荐
处理器: Pentium 600MHz or faster processor ( 推荐1GHz或以上)
操作系统：Windows® XP Service Pack 2 或以上版本
内存：256 MB of RAM (推荐512MB或以上)
硬盘空间：450 MB
支持构件：Microsoft .NET Framework 2.0 (硬盘空间200 MB~450 MB)
显示条件：VGA或以上 (640x480分辨率) (推荐1024x768)
