殘酷的幽默

1/16/08

繼團體票訂完後就是電話訂票了，雖說方式比N年前要先進不少，可還是有人得不到恩惠啊，不信的話可以摸摸訂票電話的熱度，或者關注一下廣場上排隊人群的心情。即使你訂到了票，你還得去排隊領票，暫且拋開車船勞累不表！新聞裏講：廣州火車站售票廳起沖突 3人持刀砍人1人受重傷。

盡管聽說趕集網可以買火車票，但並不是每個上網的人都會有支付工具。退一步說，同城交易對於現在的網購環境還是不太令人放心啊！

快過年了，大家都急著討要一張難得的車票（尤其是火車票），但是大家都知道買賣之間的潛規則啦。這年頭，雖說政策規定都在四處宣傳打擊倒票分子，可他們仍然”上有政策、下有對策”啊！只要鐵路線一朝存在、火車仍然在跑，吃這碗飯的”黃牛黨”是永遠不會消停的！

牢騷太甚對自己無益。還是看看大家對一些新聞事件的評論，體味一下當下社會實情吧：

1）《鹿鼎記》不符合1夫1妻制 被勒改700處(組圖)

看了廣電總局的修改意見，一種智商上的優越感猶然而生。

2）“奴工”獲救後經當地部門“轉手”被送給黑窯主

一天，我帶著我家的貓上街，去買魚，買完魚，我有事情不能回家，我就叫我家的貓把我買來的魚帶回家，放到廚房的盤子裏，貓很聽話就答應了，結果回家一看，魚？？？？？丟了，後問貓,貓說不知道,沒這回事.

3）竊鉤者誅竊國者侯

這真是殘酷的幽默！在這塊神奇的土地上，公平與正義，道德與法律還有生存的空間嗎？！

4）要死也要死在國外——也看韓國利川冷庫賠償事件

著名學者吳思一直致力於研究中國社會存在著的血酬定律：為了一定數量的生存資源，可以冒多大的傷亡風險，可以把自身生命這個資源損害到什麼程度。

5）吃打不吃記—-對抓蝦事件的幾點看法

而在中國這樣的人情社會，信用崩潰，法律無力，實名又有什麼實際意義？我上恐龍都不上你，我信ID都不信你。

推薦k-meleon瀏覽器

1/14/08

從CnBeta那裏才聽說了k-meleon這款開源的瀏覽器,英文好的可以看這裏：
http://kmeleon.sourceforge.net/

如果對Mozilla的Firefox很熟悉的話，就會對Gecko引擎核心不陌生吧？！同樣地這個k-meleon也是采用Gecko引擎，但是經過使用後發覺比Firefox2.0要快很多,配合一個loader程序讓瀏覽器開啟更迅速！特別是在設置（屬性）選項裏面的內容很豐富，更加方便的個性化瀏覽效果，還有隱私插件、廣告過濾功能。對于機器又老和網速又慢的我來說還是蠻適合的！

當然我安裝的是k-meleon CCF ME 0.08 B4 V1 中文版本，可以在下面下載最新版：
http://kmeleon.et8.net/K-MeleonCCFME0.08b4.exe

1.13日更新版本：Beta4 中文版 V2

添加了 Lua File System 部件;
屬性窗口中的皮膚列表會自動讀入皮膚目錄列表;
廣告過濾功能加入了圖片右鍵菜單中的”使用userContent.css”菜單項目，方便過濾;
userContent.css 方式的過濾加入了預覽功能，當加入一個項目後會自動顯示當前頁面過濾後的結果. (不過依然需要重新啟動瀏覽器來完成過濾設置功能);
屬性窗口常規中可以設定頁面的默認語言，默認即為中文.

k-meleon是適用于windows平臺的，但要注意她的幾個不同版本之間的小小區別哦：

1）K-Meleon是官方版http://kmeleon.sourceforge.net/

目前穩定版本為1.1.3, 采用layer插件模擬多標簽瀏覽； 支持XUL界面(支持XUL的好處:可以使用一些經過稍微改造firefox的擴展)。

2）K-Meleon目前的測試版本為1.5b22, 原生支持多標簽瀏覽。

3）K-MeleonCCF是coolwulf為K-Meleon做的帶技術預覽性質的非官方版; 基於1.5beta版本, 采用lua語言的macros, 原生支持多標簽瀏覽; 支持支持XUL界面.

4）K-MeleonCCF ME K-MeleonCCF的Non-XUL版本; 為追求極致的速度, 簡化Gecko組件, 完全去除XUL界面支持; 不支持以XUL界面為基礎的擴展(比如AdBlock plus一類的firefox擴展).

趕快加入這款具備Gecko引擎、並且快速、輕量、高度可定制化的瀏覽器行列吧！
想要知道更多的技巧或者遇到疑問，可以訪問以下論壇：

1）深度技術專欄 http://bbs.deepin.org/thread.php?fid=253
2）CCF精品技術論壇 http://bbs.et8.net/
