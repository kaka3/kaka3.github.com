Firefox代言人.機器人

4/9/08

Firefox越來越火，難怪狐迷們叫她”火狐”呢！

而關於她的神話故事也在圈中傳播開來了，不過是以“代言人”的形象出現的。（要感謝”閱微堂”給我帶來了“水木社區”上的2個鬼故事，呵呵！）

另一個更加詭異事件是Firefox被想象為”機器人”，將於2008年降臨到地球，破解這一密令通道的方法是按照“天涯博弈”所說的：在Firefox 3 Beta 5的瀏覽器地址欄輸入指令（about:robots）即可。



Firefox 3 Beta 5自開放之日起已經有幾天了，雖然修補了一些漏洞，但亦有人發現使用時頻繁出錯的現象，而Firefox 3 的最後正式版預計將在6月發布。（簡體中文版下載）

今日推薦我那天（大概4月2日）發現的一個在線瀏覽器安全測試：Browser Security Test，特此用它來測試一下Beta 4的安全性（在2號這天我還在用Beta 4的呢！），當然它也適用於IE瀏覽器。（現在還有人用IE6嗎？如果有的話，那肯定是屬於這裏所指的5種類型之一。）

信仰.農村.黑律師

4/8/08

以下是一些摘自他（她）人的言語,我自己也認為所處的現實讓人頗感無奈！

1）誠然，生活的美好、幸福是我們永遠追求的信仰。然而，倘若只能選擇或偏好一樣，我還是要我的“思考的生活”。思考也許有點累，也許不那么輕松，然而比起“信仰”的破滅與虛幻，還是要實在和可愛得多了。

——左黎《思考的生活還是信仰的生活？》

2）記得我小時候，村子裏在村委會還有個閱覽室，櫃子裏放著四大名著和一些演義類小說。現在的村委會只有一張台球桌，那些圖書早已不見其蹤。

——餘松 《農村日記（7）》

3）社會正在轉型，新生事物層出不窮，也怪象叢生。在咱們的市場經濟中，就有些怪現象，本該由市場解決的老是得依靠政府，最後導致壟斷，尾大不掉；另一些應該由政府供應的公共服務，比如免費的法律援助，則要靠市場解決……這么說來，“黑律師”也就是市場經濟不規范的產物，自然將會隨著越來越自由有序的競爭，得到妥善的解決。

——令狐補充《市場催生黑律師》

市民申請經濟適用墓地購墓資格審核指南

時間：04-04

一、購墓資格條件

廣州市中低收入家庭符合下列對象的，可以到廣州市殯葬辦填埋辦申請辦理經濟適用墓購墓資格審核。

對象：廣州市城鎮戶口，現家庭年人均身高低於 170cm，人均體重低於140斤，且未享受過買墓、海葬、網墓、經濟適用墓和參加集資建墓等購墓優惠政策的家庭。

注：身高不計穿鞋高度 體重應在飯前便後測量

二、購幕資格辦理程序

1、到廣州市殯葬辦44樓服務大廳窗口領取《廣州市市民購買經濟適用墓地資格審核表》（簡稱《審核表》）或登錄廣州市墓地局網站下載《審核表》；

2、填寫《審核表》並到申請人及其配偶所在的單位證明蓋章（待業、無單位、個體經營者及私企、外企人員到街道辦事處證明蓋章）；

3、將《審核表》及有關證明材料交回服務窗口；

4、市墓改辦填埋辦資格審核後在墓地局網站進行公示，公示期15天； 5、 25個工作日後（含公示期15天）在服務窗口領取《廣州市居民購買經濟適用墓地資格證明》（簡稱《購墓資格證明》）

三、申請資格審核須提交的相關證明材料

1、申請人及其配偶和購墓後產權共有人身份證原件及複印件；
2、戶口本原件及複印件；
3、預測死亡證明（如：健康證、或單位證明）原件及複印件；
4、家庭成員身高體重證明；
5、由單位或街道瞎扯辦提供計劃死亡證明書；
6、經閻羅王判決所死亡被收回者，需複印閻羅王判決書；
7、有關的其它資料或證明。
上只能印同一類型的證件（如全部為身份證）。

四、說明

1、經資格審核符合購墓條件的家庭取得《購墓資格證明》，《購墓資格證明》有效期為兩年，在此期間申請購墓家庭可根據廣州市經濟適用墓地的建設和推出情況，自行到各經濟適用墓銷售點選購經濟適用墓地；購墓時市殯葬辦填埋辦將對購房資格進行複核，如在獲得《購墓資格證明》後家庭身高及體重發生改變而不符合購墓條件的，將取消購墓資格。

2、經濟適用墓銷售點將根據經濟適用墓的建設情況逐步推出，並在市墓地局的網站上公布。

3、廣州市墓地局網址：請點擊進入。

4、本版內容純屬搞笑，如有雷同，那更搞笑。
